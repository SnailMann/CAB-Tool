## 介绍

`CAB Tool`  就是**csdn-article-backp-tool**，只是我有点懒，不想打这么长


- 需要登录账户密码
- 目前不支持备份图片（`下次有空再实现`）
- 目前不支持备份私密文章，只能备份自己的公开文章
- `CAB Tool` 直接爬取的是csdn markdown编辑器的文本，备份的文本与编辑器上无异
